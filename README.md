# 遺伝的アルゴリズムを用いた履修科目選択ツール
学生向けツール

## 実行環境

- Python 3.5.2 (Anaconda)
- Visual Basic for Applications (VBA)

## 必要なライブラリ

- numpy
- xlwings
- matplotlib
- os
- random

## 使い方

1. .xlsmファイルに教科名と難易度を入力する
2. ”計算開始”ボタンを押して実行
3. Resultシートが生成され、選択された科目が出力される

現在の条件設定は製作者の学校の修了条件を参考にしているため自分の学校の条件に変更する必要がある。
